🎯 Aktualny System

Ta strona działa na systemie VM

Możesz przetestować tę samą aplikację na różnych OS:

# Ubuntu 22.04 (domyślny)
dockvirt up --os ubuntu22.04

# Fedora 36
dockvirt up --os fedora36

  • ✓ Wbudowany reverse proxy (Caddy)
  • ✓ Zero-config deployment
