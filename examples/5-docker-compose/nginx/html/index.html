About This Demo

This is a demonstration of serving static content through Nginx in a Docker Compose setup, running inside a KVM virtual machine managed by dockvirt.

How It Works

  1. The main application is a Flask app running in a separate container
  2. Nginx acts as a reverse proxy, routing requests to the appropriate service
  3. Static files (like this page) are served directly by Nginx
  4. All containers run inside a KVM virtual machine for isolation
